# Project_2_Tinkering_Cam
niveau 4 AO project 2 periode 3 sectie 2 tinkering securitysysteem


// We hebben gekozen voor het project "Sectie 2: TINKERING" Security-systeem.
*Omschrijving van het project:// Met een raspberry Pi ga je een systeem opzetten dat kan functioneren
als security-camara. De functionaliteit breid je uit van beelden opnemen tot het geven van alerts
via e-mail en het aanbieden van streaming beeld bij detectie van beweging //*

> Wat we gaan doen:
We gaan een camara laten werken dat als hij iets detect (Als er beweging plaatsvind) Dat hij dán pas
opneemt, en dat er een lampje/alarm af gaat.

> Hoe gaan we dit uitvoeren?
We gaan dit uitvoeren met Python/Javascript, We gaan hiermee werken met onze eigen kennis en met
de kennis van Google.

We hebben hier 10 week de tijd voor om het project zo goed mogelijk af te krijgen.
